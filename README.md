# 🔐 Password Generator Web App

## 📌 Project Description
This is a simple and powerful Password Generator built using HTML, CSS, and JavaScript. It generates strong, random, and unique passwords based on user preferences like length, uppercase, lowercase, numbers, and special characters.

This project helps users create secure passwords instantly and improves web security awareness.

---

## 🚀 Features
- Generate strong and unique passwords
- Customize password length
- Include:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Symbols (!@#$%^&*)
- Copy password to clipboard
- Responsive and clean UI

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ⚙️ How It Works
1. Select password length
2. Choose required options (uppercase, numbers, etc.)
3. Click on "Generate Password"
4. Copy and use the password
